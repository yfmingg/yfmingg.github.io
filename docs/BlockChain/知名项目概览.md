# 知名项目概览

APR代表年度百分比率（Annual Percentage Rate）。它是实际的年回报率，不考虑复利的影响。

APY代表年溢率百分比（Annual Percentage Yield）。它是实际的年收益率，考虑了复利的影响。

APY更适合计算你的投资回报，而APR在贷款中更常见。
